# Two Leaves Tea Clone Website ☕

A modern frontend clone of the Two Leaves and a Bud tea website built using HTML and CSS.  
This project focuses on recreating the original website UI with clean layouts, typography, product sections, cards, hero banners, and responsive styling.

## 🌐 Live Inspiration

Original Website:  
https://www.twoleavestea.com/

---

## 🚀 Features

- Responsive Navbar
- Hero Sections
- Product Cards
- Review Cards
- Journal Section
- Tea Latte Showcase
- Background Image Sections
- Custom Buttons & Typography
- Modern Layout Design
- Pure HTML & CSS

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Positioning
